# 🚀 DevOps Test Platform

🔗 **Live Demo:** [https://rushi83081.github.io/test-platform/](https://rushi83081.github.io/test-platform/)

---

## 📌 Overview

A **DevOps Test Platform** built using **HTML, CSS, JavaScript** with a **serverless backend (Google Sheets + Apps Script)**.

---

## 🎯 Features

* Linux, AWS, Git, Docker, Kubernetes, Terraform, Jenkins, DevOps tests
* Timer-based quizzes ⏱️
* Random questions & options 🔀
* Instant results 📊
* Smart score update (no duplicates) 🔁

---

## 🛠️ Tech Stack

* Frontend: HTML, CSS, JavaScript
* Backend: Google Apps Script
* Database: Google Sheets
* Hosting: GitHub Pages

---

## ⚙️ How It Works

User → Takes test → Gets score → Data stored in Google Sheets (update or insert)
